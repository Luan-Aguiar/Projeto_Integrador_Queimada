Monitoramento de Queimadas

Acompanhe os dados e notícias sobre eventos de queimadas que estão ocorrendo ao redor do mundo.

Os dados são retirados da API provida pelo Programa Queimadas do INPE, que disponibiliza dados atualizados sobre os focos de queimadas no Brasil e em outros países.
